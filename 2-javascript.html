Para generar el listado utilizando handlebars, hay que dividir la información en dos. Primero, debe crearse un template de handlebars, el cual es creado utilizando un script de tipo "text/x-handlebars-template". Dicho script, va a hacer render de un objeto con la propiedad nombre ("name") y a ciclar en la propiedad jugardores ("players"), para mostrar la información del nombre ("fullName") y del número ("number") de cada uno de ellos.

Pero eso sólo define el template a mostrar. Para que aparezca la información, hay que utilizar la librería handlebars para crear un template con la variable «team», la cual va a tener la información a pasar al template. Luego se hace render del template, y se asocia al div deseado. El template se va a generar dentro del objeto seleccionado, en este caso "code-handlebars-render". La ventaja de este módelo es que el formato definido en el template se mantiene independiente de la lógica de la aplicación, permitiendo que los ajustes de interfaz no afecten la lógica de la misma.
