Ember Tutorial

Introducción

Este tutorial se va a enfocar en la creación de una aplicación HTML básica utilizando utilizando Ember.js (http://emberjs.com/).
La idea del tutorial será crear una aplicación que permita ver un listado de usuarios para un equipo en general, y se utilizará como ejemplo el "dream team" de baloncesto de EEUU que ganó la medalla de oro en la olimpiadas de Barcelona '92.

Sin embargo, cambiando los datos, puede ser utilizado para mostrar diferentes equipos de cualquier disciplina.
